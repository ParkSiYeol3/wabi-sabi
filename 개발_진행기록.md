## 2. 기술 스택

| 영역 | 채택 | 비고 |
|------|------|------|
| 프레임워크 | **Next.js 16.2.9** (App Router) | Turbopack 기본, `middleware`→`proxy` 개명, `cookies()` async 강제 |
| 런타임 | React 19.2.4 | |
| 언어 | TypeScript 5 | `src/` 디렉터리 구조 |

## 4. 현재 코드 구조 (`web/`)

```
web/
├── src/
│   ├── app/
│   │   ├── layout.tsx        # 폰트·메타데이터·AuthProvider
│   │   ├── page.tsx          # (기본 스캐폴딩 — 미수정)
│   │   └── globals.css       # 디자인토큰 (@theme)
│   ├── components/
│   │   └── auth-provider.tsx # 세션→Zustand 동기화 (env 없으면 스킵)
│   ├── lib/supabase/
│   │   ├── client.ts         # 브라우저 클라이언트
│   │   ├── server.ts         # 서버 클라이언트 (async cookies)
│   │   └── proxy.ts          # 세션 갱신 헬퍼 (env 없으면 스킵)
│   ├── store/
│   │   └── auth.ts           # Zustand 인증 스토어
│   └── proxy.ts              # Next 16 proxy 엔트리 (matcher 설정)
├── supabase/
│   ├── migrations/
│   │   ├── 0001_init.sql     # 8테이블
│   │   └── 0002_rls.sql      # RLS 정책
│   └── seed.sql              # 카테고리 4개
├── .env.local                # (gitignore) Supabase 키 — 아직 빈값
└── .env.example              # 키 템플릿
```
